Share buttons for WordPress from Yandex. 

== Description ==

<strong>Easy Yandex Share</strong> automatically adds social sharing icons at the beginning or end of a post (you can select the types of posts you want buttons to be added to in the plugin's settings)

Social networks supported:

*  Yandex.Collections
*  VKontakte
*  Facebook
*  Odnoklassniki
*  Moi Mir

Unlike other services adding sharing buttons in the script from Yandex there is no advertising and even the name "Yandex" is not mentioned anywhere.

== Frequently Asked Questions ==

= Installation Instructions =

1. Visit &#171;<strong>Plugins\Add New</strong>&#187;.
2. Search for &#171;<strong>Easy Yandex Share</strong>&#187;.
3. Install and activate the plugin.
4. That's all.

Plugin settings are available in &#171;<strong>Settings\Easy Yandex Share</strong>&#187;.

= There'e no menu in my theme. What should I do? =

Enable the &#171;<strong>Popup position</strong>&#187; option under &#171;<strong>Outside the container</strong>&#187;.

= Does the plugin support localization? =

Yes, please use [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/easy-yandex-share).
